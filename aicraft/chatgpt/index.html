Mini Minecraft

A browser-based Minecraft-style game built with HTML & JavaScript

Controls

  • WASD / Arrow keys – Move
  • Space – Jump
  • Left click – Break block
  • Right click – Place block
  • 1–6 – Select block

About

This is a 2D sandbox game inspired by Minecraft. It features procedural terrain, physics, enemies, and textured blocks — all running directly in your browser.

Tips

Make sure all texture files are inside the textures/ folder. If a texture is missing, the block will appear magenta.

Built with ❤️ using HTML5 Canvas
